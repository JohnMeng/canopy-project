Canopy Cloud Service

The Canopy Cloud Service is a server-side application written in golang that uses CassandraDB for data storage.

The Canopy Cloud Service is open source and can run in a public cloud, private cloud, LAN, or even on your laptop. However, the easiest option is to point your devices to our hosted deployment and not worry about the cloud at all!

In order to point your devices to our hosted deployment, use the following configuration settings:
{ "cloud-host" : "ccs.canopy.link", "cloud-use-https" : true, "cloud-https-port" : 433, }
