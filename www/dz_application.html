Canopy Application Development

Canopy has been designed with application developers in mind. The Canopy Cloud Service provides a REST API that will soon support OAUTH 2.0.

Additionally, application client libraries provide a simple way to interact with Canopy-enabled devices from the language of your choice.

For example, the Javascript client library lets you write code like:

myWeatherStation.properties.humidity.value();
