canopy Because everything deserves an API.


The canopy software platform gives real-world objects a representation on the Internet in just a few lines of code. It's a great tool for rapid IoT prototyping, while powerful enough to be used in massive-scale production deployments. Best of all, it's completely open source.

How it works
1.
You describe your device's capabilities using our Smart Device Description Language (SDDL).
2.
You write a small amount of embedded code, using libcanopy to send status reports to the cloud and recieve control events from the cloud.
3.
The Canopy Cloud Service talks to your device over websockets, stores historic sensor data, and translates the device's capabilities into a REST API.
4.
App developers use the canopy REST API to interact with your canopy-enabled device.
