Roadmap and Milestones

This page contains information about the release cycle, roadmap, and upcoming milestones for Canopy.

Release Process

Canopy's core components follow a three-month release cycle. Releases alternate between feature-focused (emphasis is on adding new features) and quality-focused (emphasis is on improving code quality).

The last three (3) weeks of each release cycle are dedicated to integration, testing, and critical fixes only, so plan to get your contributions in early.
