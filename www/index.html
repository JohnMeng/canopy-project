Canopy is a software framework for giving real-world objects a representation on the Internet in just a few lines of code. It's a great tool for rapid IoT prototyping, yet powerful enough to be used in massive-scale production deployments. Best of all, it's completely open source.

How it works

1.
You describe your product's capabilities using our Smart Device Description Language (SDDL).
bool on_fan_speed_control( CanopyContext ctx, uint8_t value) { /* set speed using I2C */ result = i2c_write( SPEED_PIN, value); return result; }
2.
You write a small amount of embedded code, using libcanopy to send status reports to the cloud and recieve control events from the cloud.

3.
The Canopy Cloud Service talks to your products over websockets, stores historic sensor data, and translates product capabilities into a REST API.
4.
App developers use a simple REST API to interact with your Canopy-enabled products.
