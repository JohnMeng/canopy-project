Cloud Variables

Canopy's Cloud Variables synchronize variables in firmware with variables in apps, using the cloud as a broker. This makes it easy to rapidly develop and deploy cloud-based IoT solutions.

Embedded Code (Device Firmware)

#include <canopy.h>
...
canopy_var_set_bool("toast_ready", true);


Learn more about coding with Cloud Variables 		Application or Server Code

<script src="canopy-client.js"></script>

canopy.device(id).var("toast_ready").onChange(
    function(value, error) { alert(value); }
);

Cloud Variables are extremely flexible and can be used for sensor data monitoring, remote configuration and remote control.

The Canopy Cloud automatically tracks the historical values of each Cloud Variable, so you can generate plots and perform analytics.
