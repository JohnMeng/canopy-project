You can help!

Canopy is an open source project that relies on contributions from community.


Contribute Code

If you're a developer who is considering contributing code to the Canopy project, here are some ways you can help:
  • We need more Application Client Libraries. We want it to be simple to interact with Canopy-enabled devices from any programming language. Currently, we only support Javascript and REST. We have a big wishlist: Java, Swift, Python, and Objective-C.

  • Core platform development. We're always improving the core Canopy platform with cool new features and bug fixes. Whether you're interested in wifi security, UIUX design, or something else entirely, we can find a way for you to help.
Contact us at sw@canopy.link if you're interested in contributing.
